# 4. README



## НАЗВАНИЕ

**Дизель Арена: Тренировки**

---

## ОПИСАНИЕ

Мобильное приложение для спортсменов и тренеров спортивного комплекса «Дизель Арена». Позволяет вести дневник тренировок, отслеживать прогресс, назначать и получать тренировочные программы.

---

## УСТАНОВКА

Для установки скачайте файл `.apk` из раздела **Releases**.

1. Найдите файл в файловом менеджере устройства и нажмите на него;
2. Подтвердите установку в открывшемся окне.

**Системные требования:** Android 7.0 и выше.

---

## ИСПОЛЬЗОВАНИЕ

Приложение «Дизель Арена: Тренировки» предоставляет следующие возможности:

### Основные функции

**Для спортсменов:**
- **Учет тренировок** — создание записей о выполненных упражнениях;
- **Программы** — просмотр тренировочных программ, назначенных тренером;
- **Напоминания** — уведомления о предстоящих тренировках;
- **Статистика** — отслеживание прогресса в упражнениях.

**Для тренера:**
- **Мои спортсмены** — просмотр списка подопечных;
- **Конструктор программ** — создание тренировочных программ;
- **Назначение программ** — выдача программ спортсменам;
- **Отчеты** — просмотр выполненных тренировок подопечных.

### Работа с приложением

- При запуске открывается экран входа по номеру телефона;
- Система автоматически определяет роль (спортсмен / тренер);
- Для спортсмена доступны: дневник тренировок, программы, статистика;
- Для тренера доступны: список спортсменов, конструктор программ, отчеты;
- Данные хранятся локально на устройстве.
